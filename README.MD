## iOS



### 1. [APTV](https://apps.apple.com/cn/app/aptv/id1630403500) 有内购
```
当前版本:  1.0.24
免费版: 配置只能添加1条
优点: 
1. 可显示延迟
2. EPG节目单, 支持回看
```

### 2. [Alice Player IPTV OTT](https://apps.apple.com/cn/app/id1594670709) 免费
```
当前版本： 1.4.2
存在问题：
1. 配了EPG 没展示出来
```

### 3. [~~ntPlayer~~](https://apps.apple.com/cn/app/ntplayer/id1613758141) 免费, 有广告
```
当前版本: 2.3.1 
优点: 
1. 智能合并同节目播放源 
2. 支持副屏播放
存在问题:  
1. 偶尔频道加载无数据, 收藏丢失 
2. EPG节目单没加载出来
```
